不一樣的心 - 李敏

深圳屠夫殺了十二個女孩，被拘捕後出庭時表現冷靜，受害人親屬都不明白為甚麼殺人犯可以沒半點罪咎。
常聽到人說「人同此心」，其實我不認同。
每個人的心態不一樣，就算是心跳的頻率也不一樣。
研究暴力傾向的科學家認為，心跳慢的人對於一般外來刺激的感受不大，他們需要經歷很強的刺激才興奮，換句話說，生理上心跳慢的人就是心理上膽大的人。
我的心跳頻率比一般人慢，一次做心電圖，技術員還以為機器壞了，後來醫生告訴我，心跳慢的人多是運動健兒，但我又不是。
當然，我絕不是說心跳慢就等同暴力傾向，只是研究指出有暴力傾向的人很可能是心跳慢的。
的確，有某些方面我表現得很大膽，例如半夜三更看法醫解剖書、站在台上對觀眾說話時仍能保持平常心，但對類似過山車、笨豬跳和猛鬼屋則不敢恭維，再沒有十五二十時的勇氣。
每個人的心都不一樣，就算同一個人的心都不一樣。
